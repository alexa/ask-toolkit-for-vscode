January 2021 update

Welcome to the updated Alexa Skills Toolkit for February 2021 (version 2.4). This update introduces a number of new features and enhancements.

What's new?

Support for APL 1.5

The ASK Toolkit now supports and renders APL 1.5 multimodal documents in the APL Previewer and Simulator views. Please check the What's New in APL 1.5, to get more details about APL 1.5 version.



Support for APL ExecuteCommand Directive in Simulator

The simulator will now support processing APL ExecuteCommand directives in the skill response. For more information about ExecuteCommand directive, check here.



ACDL Syntax Highlighter (Beta)

Alexa Conversations Description Language (ACDL) (Beta) is a descriptive language, to write skills using Alexa Conversations. The ASK Toolkit now supports syntax highlighting features for ACDL files. For more details, see our developer documentation about Alexa Conversations Description Language (Beta).

The ASK Toolkit has limited support for ACDL files. You will need ASK CLI to manage your ACDL skills. Please check ASK CLI setup for ACDL for more information.
