July 2021 update

Welcome to the updated Alexa Skills Toolkit for July 2021 (version 2.7). This update introduces a number of new features and enhancements.

What's new?

Support Node.js 12 for hosted skill

The ASK Toolkit now will use Node.js 12 as the default runtime for newly created Alexa hosted skill.
