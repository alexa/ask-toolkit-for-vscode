March 2023 update

Welcome to the updated Alexa Skills Toolkit for March 2023 (version 2.13). This update introduces a number of new features and enhancements.

What's new?

Language Support for Alexa Conversation Descriptive Language

You can use the Alexa Skills Kit (ASK) Toolkit for Visual Studio Code (VS Code) to edit dialogs written in Alexa Conversation Descriptive Language (ACDL). For more information, see the official technical documentation.

Support APL 2023.1

Upgrade APL Suggester and Web Viewhost to support APL 2023.1

Documentation

For more details about features, see the documents at the ask-toolkit-for-vscode Github and at the Amazon Alexa website.
